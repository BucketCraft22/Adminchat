# Adminchat

Adminchat - A minecraft plugin for your admins on VELOCITY


## About

Adminchat is a simple Velocity plugin for Minecraft servers that allows administrators to privately message each other in-game. This allows them to hold private discussions on the server without the public seeing their messages. They can see in which server is the This is a quick and handy plugin for a single, small server.

Adminchat was created by BucketCraft.

## Permissions

**`adminchat.use`**: Use the `/adminchat [message]` command or just `/adminchat` to toggle admin chat mode (/ac does the same that /adminchat is doing)
